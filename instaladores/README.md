# Instaladores

Estructura de scripts de empaquetado por plataforma:

- `windows/`: ejecutable e instalador NSIS para Windows
- `linux/`: ejecutable Linux y empaquetado `tar.gz` y opcionalmente `.deb`
- `macos/`: app de macOS y empaquetado `zip` y opcionalmente `.dmg`

Cada carpeta incluye su propio `README.md` con el comando exacto para generar los artefactos.
